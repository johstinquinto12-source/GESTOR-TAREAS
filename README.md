# 📋 Gestor de Tareas Escolar

Una Aplicación Web Progresiva (PWA) rápida, intuitiva y visualmente atractiva diseñada para ayudar a los estudiantes a organizar su semana escolar, gestionar sus tareas y calcular sus tiempos de estudio. 

## ✨ Características Principales

* 🗓️ **Creador de Horarios Personalizado:** Añade tus materias organizadas por día de la semana (Lunes a Viernes) mediante una interfaz de "píldoras" (pills) de colores generados automáticamente.
* 🗂️ **Sistema de Flashcards interactivo:** Navega por tus materias usando un carrusel táctil. Cada materia funciona como una tarjeta donde puedes:
    * Marcar si hay tareas pendientes.
    * Asignar fechas de entrega con un calendario integrado.
    * Establecer un tiempo estimado (en horas) para completar los trabajos.
    * Añadir descripciones o notas específicas.
* 📊 **Panel de Estadísticas:** Visualiza en tiempo real la cantidad de tareas totales y las horas de estudio requeridas.
* 🌓 **Modo Oscuro Integrado:** Interfaz adaptable con un botón (☀️/🌙) para alternar entre modo claro y oscuro, guardando tu preferencia automáticamente.
* 🔔 **Gestión de Notificaciones:** Cajón interactivo de notificaciones para recordatorios de tareas, configurado para reactivarse diariamente a las 6:00 AM.
* 💾 **Persistencia de Datos:** Utiliza `sessionStorage` y `localStorage` para guardar tu horario y configuraciones sin necesidad de una base de datos externa.
* 📱 **Diseño Responsive y PWA:** Interfaz fluida que simula una aplicación nativa, optimizada tanto para pantallas móviles como para computadoras de escritorio.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue desarrollado íntegramente con tecnologías web estándar, sin el uso de frameworks externos para mantenerlo ligero y rápido:

* **HTML5:** Estructuración semántica.
* **CSS3:** Variables CSS, animaciones personalizadas (keyframes), Flexbox, diseño de interfaces tipo cristal (Glassmorphism) y Media Queries.
* **JavaScript (Vanilla JS):** Lógica del DOM, manejo de eventos, gestor de estado local y cálculos de tiempo/tareas.
* **Fuentes:** [Quicksand](https://fonts.google.com/specimen/Quicksand) (Google Fonts).

## 🚀 Cómo usar este proyecto

1. Descarga o clona este repositorio en tu computadora.
2. Abre el archivo `Index.html` en cualquier navegador web moderno (Chrome, Edge, Firefox, Safari).
3. **Paso 1:** En la pantalla de inicio, haz clic en "Click para continuar".
4. **Paso 2:** Crea tu horario agregando las materias correspondientes a cada día. Puedes usar la sección "Otros" para añadir categorías personalizadas.
5. **Paso 3:** Haz clic en "Siguiente" para acceder al menú principal, donde podrás gestionar los detalles, fechas y tiempos de cada asignación en las Flashcards.

*(Nota: Al estar preparado como PWA, si cuentas con el `manifest.json` y el `sw.js`, el navegador te ofrecerá la opción de "Instalar" la aplicación directamente en tu dispositivo).*
